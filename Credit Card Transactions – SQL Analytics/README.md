# Banking Transactions Analysis Project

This project analyzes a banking transactions dataset using SQLite to extract insights about customer behavior, transaction patterns, and credit card usage.

## Dataset
- Source: Kaggle
- Table: transactions
- Columns: transaction number, credit card number, amount, merchant, transaction date

## Analysis Tasks
1. Overview metrics: total, min, max, avg transactions
2. Spending by merchant/category
3. Monthly trends
4. High-value transactions
5. Customer behavior
6. Credit card brand analysis (Visa, Mastercard, Amex, Discover)

## Skills Demonstrated
- SQL aggregations (COUNT, SUM, AVG)
- Conditional logic (CASE)
- Date manipulation (strftime)
- Business insight analysis
